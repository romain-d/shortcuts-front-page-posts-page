Very simple WordPress plugin to add in admin page screen 2 shortcuts for Front Page and Posts Page.

== Description ==

This plugin adds 2 shortcuts in administration list page ( Pages > All Pages ) to go directly to Front Page and Posts page.
Shortcuts are added only if the Front Page and Posts Page are configured in the options. ( Settings > Reading )

Github project homepage: https://github.com/romain-d/shortcuts-front-page-posts-page

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/shortcuts-front-page-posts-page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Shortcuts are added automatically!

== Frequently Asked Questions ==

= There are no shortcuts after installation =

Verify your configuration in Settings > Reading section.
